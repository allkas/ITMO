Twitter Bootstrap Свободный набор инструментов для создания сайтов и веб-приложений

Включает в себя HTML и CSS шаблоны оформления для типографии, веб-форм,кнопок, меток, блоков, навигации и прочих компонентов веб-интерфейсов, включая Javascript расширений.
Сетки - Заранее заданные размеры кнопок, которые можно сразу же использовать, например ширина колонки 140px относится к классу span2, который можно использовать в CSS описании документа.
Типографика - Описание шрифтов, определение некоторый классов для шрифтов, таких как код, цитаты и т.п.
Медиа - Представляет некторое управление изображениями и Видео.
