Языки программирования Web-страниц

HTML

(от англ. HyperText Markup Language - "язык разметки гипертекста") - стандартный
язык разметки документов во всемироной паутине. Большинство веб-страниц создаются
при помощи языка HTML (или XHTML). Язык HTML интрепритируется браузерами и
отображается в виде документа в удобной для человека форме.

CSS3

(англ. Cascading Style Sheets - каскадные таблицы стилей) -
формальный язык описания внешнего вида документа, написанного с
использованием языка разметки.
